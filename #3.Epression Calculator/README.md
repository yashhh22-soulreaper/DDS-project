‚ñ∂Ô∏è How to Run

1. **Install Python**

```bash

2. **Download Project File**

Open terminal (or CMD) in the folder and run:

When prompted, type a mathematical expression like:

```

The calculator will display the evaluated result.

5. **Exit the Program**

Type `exit` to quit the calculator.
